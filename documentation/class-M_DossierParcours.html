Class M_DossierParcours

Définit les méthodes liées à la gestion des dossiers parcours d'un patient.

Ce fichier permet de définir les méthodes de gestion des dossiers parcours d'un patient. Cette classe est nécessaire car on souhaite stocker les différentes informations sur les activités d'un patient dans son parcours. Chaque dossier parcours est composé d'une liste d'onglets, eux même composés d'une liste de champs.
